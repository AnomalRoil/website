CDF

which stands for Cryptographic Differential Fuzzer

In 2017 I’ve worked on an automated cryptographic tester called “CDF”, as in “Crypto Differential Fuzzer”, although it’s not really fuzzing any applications, but only testing cryptographic software, provided we have a common interface and comparing inputs and outputs between different implementations.

This project was then presented at BlackHat USA in August 2017 and is available on GitHub.

Future work is still planned on it, but I’ll need to find some time first.
